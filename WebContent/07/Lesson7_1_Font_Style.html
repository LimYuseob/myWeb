글꼴을 지정하는 font-family. 정의문 font-family : <글꼴명>, [<글꼴명>, <글꼴명>]

사용자가 지원되지 않는 글꼴인 경우 위에서처럼 추가 글꼴을 선언한다.
body에 글꼴을 지정하면 하위 모든 노드에 기본적으로 상속된다. 추가 font 속성
1.font-size : 값1,값2,값n.... or <절대크기><상대크기><크기><백분율>
2.font-varient(키워드 값) : normal,small-caps

3.font 속성의 값으로 위 모든걸 한번에 정의 가능함. ex>font : <font-style>,<font-varient>,<font-weight>
3_1.시스템 글꼴 확인 방법 : win Key + R(실행) --> control --> 글꼴 --> 서체명 확인

웹폰트 : 시스템 글자가 아닌 사용자가 다운로드 받아 사용하는 폰트 형태. 확장자로는 .ttf(true type..크기가 큼), .eot, .woff, woff2 등이 있다.

적용 문법으로는 @font-face{font-family : 사용 글꼴명(변수명처럼 사용); src : local('글꼴명'), url(파일명전체1) format('확장자'), url(파일명2) format('확장자'), 파일명n.....} "@" 을 이용해 소스 경로및 설정을 적용한다.

Text Style

  1. color : 표기법으론 rgb 16진수, 영문 상수값, hsl(hsla -> alpha 지정값) . 16진수 rgb(#6자리 16진수로 표기, 중복시 하나로 축약 가능), hsl 은 백분율 표기로, 둥근 색상환을 기준으로 한다.
  2. 여긴 hsl 을 적용한 style..
  3. rgb HTML
  4. RGBA CSS
  5. Text Align : 문자 정렬 속성. justify(양쪽 균등 분할)text-align
  6. 줄 간격 조절 : line-height. 문단이 두줄 이상일 경우 줄 간격 발생함. 값으로는 px, 백분율 등을 이용함. ex> 2.0 == 200%(글자 크기의 2배로 줄간격 적용) line-height
    또한, 세로로 text 정렬시 text-height 값과 같은 값으로 주면 세로 가운데 정렬을 시킬수 있다. 미정렬   정렬
  7. text-decoration : 텍스트에 줄을 표시하거나 삭제 속성..값으론 none, underline, overline   text-decoration
  8. text-shadow : 텍스트에 그림자 효과(4개의 속성을 줄수 있다. 가로거리(필수,"-" 글의 왼쪽, "+" 글의 오른쪽), 세로거리(필수,"+" 글의 아래, "-" 반대), 번짐 값("+" 모든방향으로 번짐..크게 보임), 색상(하나 이상의 색상 지정가능함))    text-shadow

목록 스타일 : list-style-type 속성을 이용하여 ul,ol 등의 항목에 style 적용.

값으로는 disc(채운원),circle(원),square(채운사각),decimal(1시작 10진수),decimal-leading-zero(01,02),lower-roman,upper-roman,none 등이 있다.
