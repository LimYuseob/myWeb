블락 레벨 요소 : 태그 사용시 한 줄을 사용하는 요소. h1, div, p 등이 이에 해당함. 예시

인라인 레벨 요소 : 컨텐츠만큼만 공간 차지함. 다른 요소 또는 같은 레벨을 중복 사용가능함.span, img, strong 등이 있다. 예시

박스모델 : 기본적으로 모든 요소를 박스 형태임. 이런 요소들을 원하는 위치하게 끔 하려면 이 모델의 원리를 파악해야함.

박스모델은 컨텐츠를 기준으로 박스와의 공간을 padding, box 의 border, box 와 box 의 간격인 Margine 으로 구성됨

Contents 영역의 크기를 지정하는 width, height 속성 : 값으로는 px, em, %, auto(default) 이다. 예시

box-sizing : 박스 모델의 크기를 계산하는 속성. 텍스트와 이미지등의 Contents 배치시 padding, border 의 값까지 일일이 계산하지 않고, 크기를 동적으로 결정시 사용됨.
종류로는 border-box(테두리까지 포함하여 width 계산, 사용하기 편리함), content-box(컨텐트 영역만 계산..default) 예시

box-shadow : 박스 모델에 그림자 효과(수평,수직 이상 필수, 흐림정도, 번짐정도,색상,inset(안쪽 그림자) 값이 있다.) 예시

border-style 속성 : 박스의 border 에 적용되는 속성, 순서는 Top --> R --> B -->L 로 이루어진다. 값으로는 none(default) 이며, hidden(감추기, table 의 border-collapse와 같음
solid, dotted, dashed, double, groove, inset, outset,ridge 등이 있다.)예시

border-width : px or 상수값 --> 값의 갯수에 따라 적용되는 면이 틀려진다. 1(4방), 2(T & B), 3(T->R->B), 4(All)예시

border-color : width 와 같은 기준으로 최대 4개의 값을 줄수 있다. 예시

border : 위의 모든 속성을 한번에 적용할 수 있는 속성.. ex>border : 3px solid blue; 예시

border-radius : 각 모서리를 둥글게 처리하는 속성 : 값으로는 px,% 로 준다. 참고로 원형으로 처리할때는 w or h / 2 의 값을준다(px기준..image w/h 없을시 백분율). 예시1 예시2

여백조절하기. Margin --> 블락요소간의 간격을 주는 속성. 값으로는 px,%,auto(default) 를 사용하고, 각 방위별 속성은 margine-top 처럼 확인할수 있다. 값1(ALL), 값2(T&B,R&L), 값3(T,R&L,B), 값4(T,L,B,R) 예시
