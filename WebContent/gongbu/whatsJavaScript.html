DOM : HTML 문서의 요소를 변경하며, 이벤트와 연동하도록 학습한다.

BOM : Browser Object Model(브라우저 내장 객체 모델을 말한다.) --> window --> document, frames[], history, location, navigator, screen 으로 계층구성이 되있으며
document 하위로는 form[], anchors[],layers[], images[] 로 계층을 이루고 있다.

  • DOM NODE 의 종류 : DOCUMENT NODE(DOM 트리의 루트 노드, HTML 문서를 나타낸다. 내부적으로는 window.document 이며, window 는 생략 가능하기에 document 로 사용함)
  • Element Node : HTML 요소를 나타내는 노드..body, a, p, script, style, html, h1 등을 가리킨다.
  • Atrritube Node : 속성노드..즉 clss="myClass" id="myId" 등..
  • Text Node : 요소내의 텍스트를 나타낸다.
  • HTML 요소 찾기(id) : document.getElementById("id") Element Obj return --> 요소 자체를 반환하기 때문에 innerHTML 을 이용해서 속성을 가져와야 한다. 단, 입력요소인 경우 value 를 이용 값 얻는다. ex>get_elem.html
  • HTML 요소 찾기("tagName") : document.getElementsByTagName("tagName") --> 모든 요소를 찾아서 배열로 리턴. Think!!!!! 특정 노드가 Text data 를 가지게 될 경우 해당 Text 는 해당 요소의 첫번째 자식 노드로 저장됨.. ex>getElemByTag.html
